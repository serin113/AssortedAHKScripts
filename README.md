AssortedAHKScripts
==================

Just a bunch of AHK scripts that I use. 'Nuff said.


**How to use**:

1. Install AutoHotkey [here](http://ahkscript.org/).
2. Modify some of the Variables if you need to.
3. Run the script.
4. Marry the script.
